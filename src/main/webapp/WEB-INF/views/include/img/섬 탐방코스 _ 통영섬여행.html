    매물도 방면
    통영여객선터미널 → 소매물도 → 통영여객선터미널
  • 사량도 방면 자세히 보기
    사량도 방면
    가오치 터미널 → 사량도(상도) → 지리산 → 최영장군 사당 → 사량도 대항해수욕장 → 가오치 터미널
  • 한산도 방면 자세히 보기
    한산도 방면
    통영여객선터미널 → 제승당 → 망산 → 통영여객선터미널
  • 욕지도 방면(여객선터미널 출발) 자세히 보기
    욕지도 방면 - 여객선터미널 출발
    통영여객선터미널 → 욕지도 → 천황산 → 해상관광 → 연화도 → 연화사 → 통영여객선터미널
  • 욕지도 방면(삼덕항 출발) 자세히 보기
    욕지도 방면 - 삼덕항 출발
    삼덕항 → 욕지도 → 천황산 → 해상관광 → 연화도 → 연화사 → 삼덕항
